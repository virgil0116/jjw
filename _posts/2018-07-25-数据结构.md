## 数据结构
***基本概括***

```
1. 线性表
2. 栈和队列
3. 串
4. 树和二叉树
5. 图
6. 查找
7. 内部排序
```

***经典算法***

```
单链表：遍历、插入、删除
循环队列：队列空、队列满的条件
二叉树：递归遍历及应用
有序表的二分法查找
快速排序
简单选择排序
```

***一般用计算机解决问题大致步骤：1.要从具体问题抽象出一个适当的数学模式，2.设计一个解此数学模型的算法；3.编出程序，进行测试、调整直至得到最终解答。***
